## APSCHE Python Internship – Task 6 Completed

**Project Title:** License Plate Recognition System

Completed Task 6 as part of the APSHE Python Internship program. This project demonstrates a basic license plate recognition system using Python. The program processes sample vehicle plate images, identifies stored plate numbers, and displays recognition results with scan history.

### Features

* License plate detection from uploaded sample files
* Recognition of 6 sample vehicle numbers
* Scan history tracking
* Search by plate number
* Menu-driven interface
* Simple Python implementation

### Technologies Used

* Python
* File handling
* Dictionaries
* Loops and conditions
* Console-based interface

### Learning Outcomes

* Worked with Python project structure
* Implemented image sample processing logic
* Improved problem-solving skills
* Built a real-time simulation of number plate recognition
